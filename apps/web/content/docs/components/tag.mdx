## API Reference

### Tag

Container element for the tag.

| Prop       | Type                       | Default    |
| ---------- | -------------------------- | ---------- |
| `variant`  | `"stroke" \| "lighter"`    | `"stroke"` |
| `disabled` | `boolean`                  | `false`    |

### TagDismiss

Dismiss button rendered inside a tag. Uses a built-in close icon.
