## API Reference

### Toggle

Styled wrapper for `Toggle.Root` from Base UI.

| Prop      | Type                       | Default     |
| --------- | -------------------------- | ----------- |
| `variant` | `"primary" \| "ghost"`     | `"primary"` |
| `size`    | `"sm" \| "md" \| "lg"`    | `"md"`      |

All props from [Base UI Toggle](https://base-ui.com/react/components/toggle#api-reference) are also supported.
