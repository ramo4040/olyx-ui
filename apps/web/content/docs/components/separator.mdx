## API Reference

### Separator

Styled wrapper for `Separator` from Base UI.

| Prop          | Type                                                  | Default        |
| ------------- | ----------------------------------------------------- | -------------- |
| `variant`     | `"line" \| "solid-text" \| "content" \| "underline"` | `"line"`       |
| `orientation` | `"horizontal" \| "vertical"`                          | `"horizontal"` |
