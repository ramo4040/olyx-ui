## API Reference

#### Input

Styled wrapper for `Input` from Base UI.

All props from [Base UI Input](https://base-ui.com/react/components/input#api-reference) are supported.

## Examples

For accessible labelling and validation, prefer using the `Field` component to wrap inputs. See [related examples](/docs/components/field).
