## API Reference

This is a custom component, not a Base UI wrapper.

### Skeleton

A simple div with animated pulse effect for loading states. Apply sizing via inline styles or className.

| Prop      | Type      | Default |
| --------- | --------- | ------- |
| `rounded` | `boolean` | `false` |
