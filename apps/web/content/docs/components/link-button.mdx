## API Reference

#### LinkButton

Styled anchor element with variant, size, and underline options.

| Prop        | Type                                         | Default   |
| ----------- | -------------------------------------------- | --------- |
| `variant`   | `"gray" \| "black" \| "primary" \| "error"` | `"black"` |
| `size`      | `"sm" \| "md"`                               | `"md"`    |
| `underline` | `boolean`                                    | `true`    |

All props from `React.ComponentProps<'a'>` are also supported.
