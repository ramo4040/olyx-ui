## API Reference

### RatingReview

Read-only rating display. Uses HugeIcons internally for star and heart icons.

| Prop      | Type                | Default  |
| --------- | ------------------- | -------- |
| `variant` | `"star" \| "heart"` | `"star"` |
| `value`   | `number`            | `0`      |
| `max`     | `number`            | `5`      |
