## API Reference

#### Button

Styled wrapper for `Button` from Base UI.

| Prop      | Type                                            | Default     |
| --------- | ----------------------------------------------- | ----------- |
| `variant` | `"primary" \| "neutral" \| "error" \| "link"`   | `"primary"` |
| `mode`    | `"filled" \| "lighter" \| "ghost" \| "stroke"`  | `"filled"`  |
| `size`    | `"sm" \| "md" \| "lg"`                          | `"md"`      |
| `asIcon`  | `boolean`                                       | `false`     |
| `asRtl`   | `boolean`                                       | `false`     |

All props from [Base UI Button](https://base-ui.com/react/components/button#api-reference) are also supported.
